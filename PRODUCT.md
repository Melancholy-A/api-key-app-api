# Product

## Register

product

## Users

People who want a private, installable mobile AI assistant that talks to their own GPT-compatible API provider. They use it on phones and tablets for chat, search, image input, image generation, Office file work, and follow-up questions across long conversations.

## Product Purpose

Codex Mobile is a task-focused AI assistant shell. It should make advanced model features feel approachable on Android without exposing unnecessary provider complexity during normal chat.

## Brand Personality

Calm, capable, precise.

## Anti-references

Avoid decorative landing-page styling, loud gradients, heavy glassmorphism, oversized rounded cards, inconsistent icon styles, and settings screens that expose every advanced option at once.

## Design Principles

1. Keep the chat surface quiet so the answer is the interface.
2. Use familiar mobile patterns before inventing new controls.
3. Reveal advanced provider settings only when they are relevant.
4. Prefer subtle state feedback over decorative motion.
5. Preserve speed and readability over visual effects.

## Accessibility & Inclusion

Use strong text contrast, touch targets around 44 dp, readable Chinese and English typography, clear disabled and loading states, and reduced-motion fallbacks for animated UI.
